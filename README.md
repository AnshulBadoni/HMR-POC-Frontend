# HRMS Lite - Frontend

A clean and modern web interface for managing employees and tracking attendance. Built with React and styled with Tailwind CSS.

## What it does

This is the user-facing part of HRMS Lite. It provides:

- **Dashboard** - Quick overview of your team's attendance
- **Employee Management** - Add and manage employee records
- **Attendance Tracking** - Mark and view daily attendance

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool (super fast!)
- **Tailwind CSS** - Styling
- **React Router** - Navigation
- **Axios** - API calls
- **Lucide React** - Beautiful icons
